# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
|  0.1.0  | :white_check_mark: |


## Reporting a Vulnerability

If you detect a vulnerability in this codebase please open an issue with the following data.
1. A breif description of the bug/vulnerability.
2. Instructions on how to re create the bug.
3. Your operating system.
4. Other enviromental variables (if applicable).
5. Any other information that you deem useful
